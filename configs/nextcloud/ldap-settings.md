# LDAP/AD интеграция - Nextcloud конфигурация

## Настройки на връзката
- **Host:** ldaps://192.168.20.10
- **Port:** 636 (LDAPS - криптирана връзка)
- **Base DN:** DC=ict,DC=local
- **Bind потребител:** nextcloud_ldap@ict.local

## Филтриране на обекти
- Импортират се само реални потребителски профили (класове `user`, `person`)
- Login Attribute: `sAMAccountName`

## Групи
- Автоматично изтегляне на Active Directory Security Groups
- Свързващ атрибут: `member (AD)`

## Забележка за сигурност
В разширените настройки временно е изключена задължителната валидация 
на SSL сертификата, тъй като средата използва вътрешен self-signed 
сертификат (затворена лабораторна мрежа, не публична продукционна среда).
